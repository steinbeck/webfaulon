Create README.md with the following sections (in order):

1. **Title and badges:** `# WebFaulon` with a one-line description: "Interactive web demonstration of Faulon's simulated annealing algorithm for exploring the space of constitutional isomers."

2. **Demo link:** Prominent link to `https://steinbeck.github.io/webfaulon/` — use a bold "Try it live" callout. Mention it runs entirely in the browser (no server needed).

3. **About:** 2-3 paragraphs explaining:
   - What the tool does (students enter a molecular formula, configure SA parameters, watch the algorithm explore constitutional isomer space optimizing the Wiener Index)
   - The educational purpose (making the abstract algorithm from the paper tangible and interactive)
   - Based on Faulon's 1996 paper (full citation)

4. **Features:** Bulleted list:
   - Real-time SA execution with live Wiener Index chart (inspired by Figure 4 of the paper)
   - 2D molecular structure rendering via RDKit.js
   - Configurable SA parameters (temperature, cooling schedule, steps, cycles)
   - Minimize or maximize Wiener Index
   - Preset molecules for quick exploration (hexane isomers, octane isomers, decane isomers, monoterpene isomers)
   - Responsive design for classroom projection and mobile devices
   - Runs entirely in the browser using Web Workers (no server required)

5. **Tech Stack:** Brief table or list:
   - TypeScript + Vite (build tooling)
   - Alpine.js (reactive UI)
   - Chart.js (live optimization chart)
   - RDKit.js (WASM cheminformatics — 2D rendering, SMILES generation)
   - Web Workers + Comlink (non-blocking SA execution)

6. **Getting Started (Development):**
   ```
   git clone https://github.com/steinbeck/webfaulon.git
   cd webfaulon
   npm install
   npm run dev
   ```
   Then open http://localhost:5173

7. **Building for Production:**
   ```
   npm run build
   npm run preview
   ```

8. **Running Tests:**
   ```
   npm test
   ```

9. **How It Works:** Brief explanation (3-5 sentences) of the SA algorithm as implemented:
   - Starts from a random connected molecular graph satisfying the formula
   - Applies Faulon's displacement operations (equations 7-11) to modify bonds while preserving valence
   - Evaluates the Wiener Index (sum of shortest path distances) as the objective function
   - Accepts or rejects moves via the Metropolis criterion with configurable cooling schedule

10. **Citation:** BibTeX entry for:
    Faulon, J.-L. "Stochastic Generator of Chemical Structure. 2. Using Simulated Annealing To Search the Space of Constitutional Isomers." J. Chem. Inf. Comput. Sci. 1996, 36, 731-740. DOI: 10.1021/ci950093o

11. **License:** State "MIT" (or note if no license file exists yet — just add a brief "MIT License" line).

Writing style: Clear, academic but accessible. No emojis. Suitable for a cheminformatics audience (professors, students, researchers).
